# INTERRUPT-HANDLING #
Thực hiện xử lý ngắt trên lớp Device Drive và cho chạy trên Raspberry Pi 3 model B+
Nhóm 5 gồm sinh viên
-------------------------------------------------------------------------------------------------------
Đề tài: Xử lý ngắt
Mô tả: 
- Nhập "start" vào user space, 3 LED sáng tuần tự  
- Xử lý ngắt: Khi nhấn nút nhấn 3 LED tắt, thả nút nhấn 3 LED sáng tuần tự bình thường  
- Nhập "stop" vào user space, 3 LED tắt
